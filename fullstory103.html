Story 103 – The Time Machine If I build a time machine, I will visit the future. If the machine works, I will see flying cars and robot cities. If I went back to the Victorian era, I would wear a top hat. If people saw my smartphone, they would think it was magic. If the scientist had calibrated the dials, he would have arrived in the Jurassic period. If he had met a T-Rex, he would have been terrified. I would be happy if I could travel through time. If you choose a destination, where will you go? If we had used more energy, the jump would have been successful.

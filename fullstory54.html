Story 54 – The Island of Wishes If I had a boat, I would sail to the secret island. If you come with me, we will find the treasure. If we had started earlier, we would have seen the sunset. If the wind is too strong, the trip will be dangerous. I would be very happy if I found the magic tree. If the birds screamed, it would mean that a storm was coming. If you had ever been there, you would know the truth. If I had a wish, I would ask for peace. If we leave now, we will arrive there by noon.

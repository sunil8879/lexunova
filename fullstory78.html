Story 78 – The Forest Path If we walk fast enough, we will reach the cabin before dark. If the rain stops, we will have a campfire tonight. If I found a map, I would show you the way. If the bears came near the camp, the dog would bark loudly to warn us. If they had taken the left turn, they would have found the hidden waterfall. If the stars were visible, we could navigate using the North Star. I would be terrified if I saw a ghost in these woods. If you listen carefully, you will hear the sound of the river. If we had brought more food, we wouldn't be hungry now.
